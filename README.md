# PAF Project

PAF Project is a Smart Campus Operations Hub built with Spring Boot and React to manage facility booking, maintenance tickets, notifications, and role-based access.

## Project Structure
- backend/ - Spring Boot REST API
- frontend/ - React client application
- docs/ - diagrams, screenshots, and report assets
